- bash

- ssh
- scp
- find

## Exercises

- make some files, move them around, delete them
- how to get information (who/where you are, files and their attributes, env variables, command help) 
- globbing
- permissions
- pipes
- IO redirection
- scripts
- variables
- loops (for/while)
- command substitution
